# GenAI Pipeline Setup

## Files

### you have to create a `.env` file in the folder *genai* (see the exemple in *.env.example*) :

- HF_TOKEN : Acces token from HuggingFace
- GH_USER : Github username
- GH_TOKEN : Github user token


## How to run ?

- You have to run the server file *genai/app.py* on calypso server using this command(in genai directory) :

**`uv run app.py`**

- You have to run the file *genai/client.py* on your computer using this command (in genai directory) : 

**`uv run client.py`**
